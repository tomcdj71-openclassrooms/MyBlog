# MyBlog

## Pre-requesites : 
---
- PHP 8.2 ('intl' extension enabled)
- Composer
- npm (I used pnpm)

`composer install --no-dev --optimize-autoloader`

then, navigate to public and do :
`pnpm install`

if you don't have pnpm, run `npm install`
